# 🗃️ Backup Auto (Windows 10)
**Sauvegarde automatique de disque à disque**

[![Batch](https://img.shields.io/badge/Batch-Script-blue)](https://en.wikipedia.org/wiki/Batch_file)
[![Licence](https://img.shields.io/badge/Licence-MIT-green)](https://github.com/Flavien-Audinet/Backup-auto-win10/blob/main/LICENCE)

---
## 📌 **Description**
Script Batch pour **copier automatiquement** les données d’un disque source vers un disque de destination.
**Dernière mise à jour** : 3 mars 2024

---
## 📁 **Fichiers**
- `Backup Auto.bat` : Script principal

---
## ⚙️ **Personnalisation**
Modifiez le script pour définir :
- Le **disque source**
- Le **disque de destination**

---
## 🚀 **Utilisation**
1. Ouvrez `Backup Auto.bat` avec un éditeur de texte
2. Modifiez les lettres de lecteurs (ex: `C:` → `D:`)
3. Exécutez le script en tant qu’administrateur

---
## ⚠️ **Avertissement**
> ⚠️ **Vérifiez les chemins** avant de lancer la sauvegarde pour éviter les erreurs.
